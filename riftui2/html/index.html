RULES
1) Ο server αυτός είναι role play
Αναζητούμε άτομα με όρεξη,ευφάνταστα και δημιουργικά σενάρια για ROLEPLAY και όχι άτομα που θέλουνε να παίξουνε [GTA online]. Αυτό που ζητούμε από σας πριν μπείτε στον server είναι να έχετε διαβάσει και κατανοήσετε τα rules. Είναι πολύ σημαντικό να γνωρίζετε τους κανόνες γιατί έτσι μπορείτε να αποφύγετε τυχόν λάθη μέσα σε ένα σενάριο rp. Το voice chat είναι υποχρεωτικό. Αυτό που θα θέλαμε είναι να ακούγεστε σωστά και καθαρά. Το voice changer είναι επιτρεπτό σε λογικά πλαίσια. Εικόνα

2) Ευγένεια/Σεβασμός
Τα δυο πιο σημαντικά πράγματα στον server. Δεν θέλουμε σε καμία περίπτωση οι καινούριοι παίχτες του server να δέχονται[bullying]. Επ' ουδενί η παλαιότητα σας στον server δεν σας δίνει το δικαίωμα να χρησιμοποιείτε ακατάλληλη φρασεολογία και να γίνεστε αγενής. Αν πέσει στην αντίληψη μας κάτι ανάλογο θα υπάρξουν σοβαρές κυρώσεις. Τέλος θα κάνουμε μια τοποθέτηση πάνω στο θέμα του σεβασμού. Όπως καταλαβαίνετε στον server θα υπάρξουν και άτομα μεγαλύτερης ηλικίας, πράγμα που σημαίνει ότι ακόμα και σε σενάριο rp θέλουμε να είστε προσεκτικοί στο τι λέτε και τι κάνετε. Εικόνα

3) OOC (Out of character)
Απαγορεύεται να βγαίνετε εκτός χαρακτήρα μέσα σε ένα σενάριο. Συνεχίστε το rp μέχρι τέλους και ας υπάρχουν λάθη. Όταν τελειώσει το σενάριο μπορείτε να κάνετε ένα report όπου θα πρέπει να εξηγείτε τους λόγους που σας οδήγησαν στο να κάνετε το report. Τα reports no reason θα αγνοούνται. Τέλος αν χρειαστείτε πληροφορίες σχετικά με το τεχνικό κομμάτι του server θα σας παρακαλούσα να το κάνετε μέσω discord ζητώντας τη βοήθεια κάποιου ενεργού staff και όχι μέσω των ingame players. Εικόνα

4) Power Gaming
Δεν θα γίνετε αποδεκτό κανενός είδους power gaming,όπως το να τρέχεις με ιλιγγιώδη ταχύτητα σε μια καταδίωξη και να τρακάρεις κατά εξακολούθηση να κατεβαίνεις από το όχημά σου και να συνεχίζεις σαν να μην έχει συμβεί τίποτα. Εικόνα

5) RDM/VDM
Απαγορεύεται να σκοτώνεις με κάθε είδους τρόπο τους συμπολίτες σου χωρίς να έχει προηγηθεί σενάριο γνωστό ως RDM/VDM. Απαγορεύεται να κλέψεις κάποιον εφόσον συνεργαστεί και στην συνέχεια να τον σκοτώσεις! Θα θεωρηθεί RDM. Εικόνα

6) Value of Life
Οφείλεται να σέβεσαι και να εκτιμάς τη ζωή του χαρακτήρα που υποδύεσαι σαν να έχεις μόνο μία. Εικόνα

7) New Life Rule
Σε περίπτωση που για τον οποιοδήποτε λόγο καταλήξεις νεκρός η μνήμη σου θα έχει υποστεί ολική διαγραφή ούτως ώστε να αποφευχθεί το ενδεχόμενο εκδίκησης. Ο μόνος τρόπος επιδίωξης εκδίκησης είναι κάποιος μάρτυρας στο συμβάν ο οποίος θα σου δώσει πληροφορίες για ότι συνέβη χωρίς να σου εξηγεί τις συνθήκες κάτω από τις οποίες βρέθηκες νεκρός. Εικόνα

8) Cop Baiting / EKAB Baiting
Απαγορεύεται αυστηρά να ενοχλείς το σώμα της αστυνομίας ή το κρατικό νοσοκομείο της πόλεως προκαλώντας τους είτε ενοχλώντας τους είτε θέτοντας σε κίνδυνο τη σωματική τους ακεραιότητα [ΧΩΡΙΣ ΟΥΣΙΩΔΗ ΛΟΓΟ]. Απαγορεύεται να κλέψεις/χρησιμοποιήσεις οχήματα της αστυνομίας και του ΕΚΑΒ . Παρά μόνο εάν έχει προηγηθεί σενάριο RP. Εικόνα

9) Meta Gaming
Δεν μεταφέρουμε οποιαδήποτε πληροφορία αφορά το role play των παικτών του game μέσω discord-stream διότι έτσι χαλάει το content-σενάριο που έχει δημιουργήσει κάθε παίχτης. Θα υπάρξουν σοβαρές επιπτώσεις! Εικόνα

10) Combat log
Aπαγορεύεται το [combat log].Δηλαδή την στιγμή που γίνεται ένα σενάριο rp να αποχωρήσεις από τον server.[Αυτό ισχύει είτε είσαι ζωντανός είτε νεκρός].
